Nightmare
recursion

The team of FunLime presents the game "Nightmare recursion". The game takes place in the nightmare of the main hero-programmer. The game has 3 levels. In each level you will encounter obstacles, such as spikes.One of the main and frequent obstacles is a recursive room. The recursive room is a room with doors located one opposite the other, so that moving forward, you can eventually come to the starting position.
